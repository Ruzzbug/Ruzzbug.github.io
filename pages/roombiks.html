Roombik's Interface

- A holographic room customisation interface for use aboard a spaceship

Role(s)

Interaction Design, User Experience, Prototype, Rooms, User Research and Testing

Software

Figma, Unreal Engine, Balsamiq


Given a brief where we had to find a way to create interfaces for individuals partaking in long distance space travel, my team and I began our research, and then iterative development. Creating a high fidelity prototype of the Roombik's cube interface. Roombik's allows the user to preview and customise their room in real time.
